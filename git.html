PASO A PASO PARA UTILIZAR
GIT IGNORE

PASO 1

Para empezar, ve a la raíz de tu proyecto, donde se encuentra la carpeta .git, y crea un nuevo archivo llamado .gitignore

PASO 2

Abre el archivo .gitignore en un editor de texto y escribe el nombres de los archivos que deseas que Git ignore.
Por ejemplo, para ignorar todos los archivos con la extensión.log, puedes añadir *.log.
Para ignorar carpetas completas escribe nombre_de_carpeta/
Y para archivos específicos escribe notas.txt`.

PASO 3

Finalmente, guarda los cambios en el archivo .gitignore. Luego, usa el comando git status para verificar que los archivos o directorios especificados no aparezcan en la lista de cambios pendientes, lo que confirmará que están siendo ignorados correctamente.
